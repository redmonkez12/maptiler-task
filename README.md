## Maptiler task

* GraphQL Crud app

### Technologies
* Bun, Nextjs, MUI, GraphQL, Tanstack store, Postgres

### Pre installation
* Add .env file to apps/maptiler
* Add DATABASE_URL (postgres) and NEXT_PUBLIC_API_URL (graphql)

### Installation
* [Bun](https://bun.sh/)
* Run docker-compose -f docker-compose.yml up -d
* Run bun postinstall
* Run bun db:push
* Run bun db:seed
